Perguntas frequentes

Dúvidas sobre integrações

O que é MCP?

É um padrão aberto para conectar ferramentas e sistemas a agentes de IA. Na prática, é o que permite o seu Agente usar buscas, planilhas, CRMs e o seu próprio sistema sem uma integração feita à mão para cada um.

É seguro conectar meus sistemas?

Sim. As conexões usam login seguro (OAuth), os acessos ficam criptografados e isolados por conta, e você concede só as ferramentas que quiser. Suas credenciais nunca são expostas ao modelo.

Consigo conectar qualquer API ou webhook?

Sim. O Agente tem uma ferramenta para acessar APIs externas com trava de segurança, então dá para integrar com quase qualquer serviço — inclusive um sistema próprio.

O Agente pode fazer coisas sem eu ver?

Só o que você permitir. Para ações sensíveis, ele pausa e pede aprovação — você aprova, edita ou recusa antes de executar.

Preciso de um desenvolvedor?

Muitas integrações são prontas e você conecta pela tela. Para casos mais específicos, um desenvolvedor ajuda — mas o básico você resolve sozinho.
