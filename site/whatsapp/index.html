Perguntas frequentes

Dúvidas sobre o WhatsApp

QR Code ou API oficial: qual escolher?

Para começar rápido e usar o seu número atual, o QR Code resolve. Para volume maior, contato ativo com modelos aprovados e a via oficial da Meta, use a API oficial (Cloud API). Dá para trocar depois.

Posso usar o número que já tenho?

Sim. Pelo QR Code você conecta o número que já usa. Na API oficial, o número é habilitado na plataforma da Meta.

O Agente passa a conversa para uma pessoa?

Sim. Quando faz sentido, ele transfere para um atendente humano — na mesma conversa, sem perder o histórico. Você assume, responde e depois devolve o atendimento para o Agente.

Consigo mandar mensagem para vários clientes de uma vez?

Sim. Você sobe uma lista em CSV e envia mensagens personalizadas (mail-merge). No canal oficial, mensagens de contato ativo usam modelos aprovados pela Meta. As respostas caem direto no Agente.

Quanto custa?

O atendimento usa os seus créditos, como no chat e na voz. Os disparos ativos são cobrados por mensagem enviada, de forma transparente. Tudo dentro do plano de R$25/mês que vira créditos.
